Alternate If Statment

I use if statments often, but last week I stumbled across a different way off thinking about this. If, for example, you wanted to check a value and depending on it return a different value, you could use a number of if statments like below.

                    let x = 5;
                    if(x === 5){
                        console.log("The value is 5.");
                    } else if(x === 6){
                        console.log("The value is 6");
                    }

This works, but it looks ugly, and is hard to deal with. Here is another to set it up..

                    let x = 5;
                    let values = {
                        5: "The value is 5.",
                        6: "The value is 6."
                    }

                    for(property in values){
                        if(property === x){
                            consolelog(values[property]);
                        }
                    }

Javascript Sort

To sort an array from largest to smallest, I see the below a lot on Google. 

                [34, 24, 29, 3].sort(function(a, b){return b - a});

After Joe Shepherd, the NSS instructor explained it, it looks like .sort feeds 34 and 24 as a and b, and returns the subtracted value of 10. 10 would mean that the second number is smaller than the first, so 24 would go after 34. It repeats this until the numbers are greatest to smallest. The next two numbers would be 24 and 29, which would return -5, so .sort would put 29 before 24.
